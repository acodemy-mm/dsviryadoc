# Changelog

All notable changes to the Virya Design System documentation site.

## [0.2.0] - 2026-06-30

### Added
- Landing page with getting started, components, and tokens quick links
- Global search command palette (⌘K)
- Mobile sidebar drawer navigation
- Live component previews with variant controls (Button, Input, Badge)
- Component detail tabs: Preview, Usage, Props, Code, Accessibility
- Breadcrumbs and sticky table of contents on doc pages
- Brand switcher on tokens page with copy-to-clipboard for all tokens
- MM (Myanmar) typography locale tab
- Documentation pages CMS (`ds_pages`) with Getting Started, Foundations, Patterns, Resources
- Do/Don't callout blocks in markdown
- Changelog page
- Figma "Open in Figma" links on component pages
- Token sync script (`npm run sync:tokens`)
- Foundation pages: Spacing & Layout, Elevation, Motion
- Pattern pages: Tables, Navigation, Empty States

### Changed
- Replaced bare `/` redirect with full landing page
- Unified docs layout with shared sidebar across components, tokens, docs, and changelog

## [0.1.0] - Initial release

- Component gallery with Supabase CMS
- Design tokens page
- Admin dashboard with CRUD
